# Portfolio Frontend

React + Vite portfolio site and admin dashboard.

## Setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Scripts

- `npm run dev` — dev server (:5173)
- `npm run build` — production build → `dist/`
- `npm run imgbb:upload` — upload images to ImgBB

## Env (`.env.local`)

See `.env.example` — Firebase client keys, `VITE_API_URL`, `VITE_ADMIN_EMAIL`, ImgBB key.
